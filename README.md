# 🌿 Streamlit Mini Apps Project

Streamlit과 생성형 AI를 활용해  
짧은 기간 동안 다양한 미니 웹 앱을 구현해본 개인 실습 프로젝트입니다.

---

## ⏱ 프로젝트 정보
- 진행 기간: 3일  
- 형태: 개인 실습 프로젝트  
- 목적: Streamlit 기본 사용법과 UI 구성 경험

---

## 📂 파일 구성

- **1.py**  
  Streamlit으로 화면에 간단한 인사 메시지를 출력하는 기본 예제
- **2.py**  
  CSS 스타일링과 레이아웃을 적용한 개인 소개용 디지털 페르소나 페이지
- **3.py**  
  하루의 감정과 한 줄 회고를 기록하고, 감정 흐름을 시각화하는 회고 다이어리 앱
- **4.py**  
  키보드 입력으로 조작하는 간단한 공룡 점프 미니 게임

---
